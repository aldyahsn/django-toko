# Django 3 project template

This is a simple Django 3.0+ project. 

## Features

- Django 3.0+
- Uses [Pipenv](https://github.com/kennethreitz/pipenv) - the officially recommended Python packaging tool from Python.org.
- Development,Production settings with [django-configurations](https://django-configurations.readthedocs.org).
- PostgreSQL database support with psycopg2.

## How to install

```bash
wget -N -q  https://raw.githubusercontent.com/koderstory/djangotemplate/main/installer-django.sh && chmod u+x .installer.sh && ./.installer.sh
```
